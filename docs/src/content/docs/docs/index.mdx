Orchestra launches isolated, time-limited workshop sessions — RStudio,
JupyterLab, or any container — on Kubernetes, with sign-in and cleanup handled
for you. It's the platform behind Bioconductor workshops at conferences and
courses.

## What Orchestra is good for

- **Teaching & workshops** — bioinformatics courses, data-science bootcamps, conference tutorials.
- **Research** — reproducible analysis sessions and collaborative environments.
- **Any stack** — point a template at any container image; it isn't limited to a single field or toolchain.
